# Касса
![](https://github.com/smpb05/DSS-Retail/blob/project-screenshots/%D0%BA%D0%B0%D1%81%D1%81%D0%B0.png)

Чтобы внести деньги в кассу, выберите операцию [Пополнение кассы](https://github.com/smpb05/DSS-Retail/wiki/%D0%9F%D0%BE%D0%BF%D0%BE%D0%BB%D0%BD%D0%B5%D0%BD%D0%B8%D0%B5-%D0%BA%D0%B0%D1%81%D1%81%D1%8B).

Чтобы записать выдачу денег из кассы, выберите операцию [Изъятие денег из кассы](https://github.com/smpb05/DSS-Retail/wiki/%D0%98%D0%B7%D1%8A%D1%8F%D1%82%D0%B8%D0%B5-%D0%B4%D0%B5%D0%BD%D0%B5%D0%B3-%D0%B8%D0%B7-%D0%BA%D0%B0%D1%81%D1%81%D1%8B).

Чтобы внести оплату по задолженности поставщику, выберите операцию [Оплата поставщику](https://github.com/smpb05/DSS-Retail/wiki/%D0%9E%D0%BF%D0%BB%D0%B0%D1%82%D0%B0-%D0%BF%D0%BE%D1%81%D1%82%D0%B0%D0%B2%D1%89%D0%B8%D0%BA%D1%83).

Чтобы внести оплату за задолжнность от клиента, выберите операцию [Оплата от клиента](https://github.com/smpb05/DSS-Retail/wiki/%D0%9E%D0%BF%D0%BB%D0%B0%D1%82%D0%B0-%D0%BE%D1%82-%D0%BA%D0%BB%D0%B8%D0%B5%D0%BD%D1%82%D0%B0).

Для возвращения в [[Главное меню]] нажмите ![](https://github.com/smpb05/DSS-Retail/blob/project-screenshots/%D0%BA%D0%BD%D0%BE%D0%BF%D0%BA%D0%B0%20%D0%BD%D0%B0%D0%B7%D0%B0%D0%B41.png)
